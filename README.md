# ☕ Core Java — Complete Fundamentals Course

A comprehensive **Core Java training repository** covering fundamental to intermediate Java concepts including OOP, collections, exception handling, multithreading, and file I/O.

---

## 📖 Overview

This repository contains hands-on Java exercises covering the essential concepts needed for software development and test automation. Topics progress from basic syntax and control structures to advanced object-oriented programming.

---

## 📚 Topics Covered

- **Basics** — Variables, Data Types, Type Casting, Operators
- **Control Flow** — if/else, switch, Loops (for, while, do-while, for-each)
- **Strings** — String methods, StringBuilder, StringBuffer, Immutability
- **Arrays** — Single & multi-dimensional arrays, operations
- **OOP Concepts** — Classes, Objects, Constructors, `this` keyword
- **Inheritance** — Single, Multilevel, Hierarchical, `super` keyword
- **Polymorphism** — Method Overloading, Method Overriding
- **Abstraction** — Abstract classes, Interfaces
- **Encapsulation** — Access modifiers, Getters/Setters
- **Collections** — ArrayList, HashMap, HashSet, LinkedList
- **Exception Handling** — try/catch, finally, custom exceptions
- **File I/O** — Reading and writing files
- **Multithreading** — Threads, Runnable, synchronization

## 🚀 Getting Started

### Prerequisites
- Java JDK 11+
- Maven 3.x
- Any IDE (IntelliJ IDEA / Eclipse)

### Installation
```bash
git clone https://github.com/Shubham00117/Core-Java.git
cd Core-Java
mvn clean install
```
